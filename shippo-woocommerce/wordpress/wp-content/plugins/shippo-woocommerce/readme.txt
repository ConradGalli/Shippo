==== Woocommerce Shippo Shipping Tool - Shippo Integration for WooCommerce by Wanderlust Web Design ====

Requires at least: 3.0
Tested up to: 3.6
Stable tag: 3.0


License: GPLv2 or later This plugin allows your users to interface with the fantastic Shippo Shipping API. 


== Description ==

Shippo Shipping Solution is a simple shipping API for your app or site. We believe in making purchasing labels, tracking packages, checking rates, verifying addresses, and all other shipping related functions as easy as possible.



== Installation ==

1. Install the Woocommerce Shippo Shipping Tool Plugin 
2. Activate the plugin
3. Go to the WooCommerce Easypost Config 
4. Check the Enable/Disable Checkbox.
5. Enter the settings that you would like you to use
6. Enter the dimensions of the products, (Easypost uses this to determine suitability of products)
